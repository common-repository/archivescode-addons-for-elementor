Awesome addons collections for Elementor page builder.

== Description ==

This plugin adds new elements/widgets to Elementor Page Builder.

For more details and demo check our official site [archivescode](http://archivescode.;/.com/archivescode-for-elementor)

Current Addons/widgets Elements:

1. slider for elementor
	* custom animation per slider
	* custom animation per caption, description, button
	* custom background image, color
	[demo](http://archivescode.com/archivescode-for-elementor/slider-for-elementor)
2. Fancy Flipbox for elementor
	* Flip transition
		* Direction up
		* Direction down
		* Direction left
		* Direction right
	* slide transition
		* Direction up
		* Direction down
		* Direction left
		* Direction right
	* Push transition
		* Direction up
		* Direction down
		* Direction left
		* Direction right
	* Fade transition
	* Zoom in trasition
	* Zoom out transition
	[demo](http://archivescode.com/archivescode-for-elementor/flipbox-for-elementor)
3. Fancy Pricing Table For Elementor
	* 3 style Header
		1. text
		2. icon
		3. custom
	* ribbon support
	* custom background Header with image
	* custom background image, gradation, custom color
	* unlimited featured list
	* custom price / text
	* custom currency
	[demo](http://archivescode.com/archivescode-for-elementor/pricing-table)
4. Many more elements coming soon…

Note: This plugin is an addon of Elementor Page Builder [Elementor page builder](https://wordpress.org/plugins/elementor/) and will only work with Elementor Page Builder installed.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

MINIMUM REQUIREMENTS

* WordPress 4.5 or greater
* PHP version 5.4 or greater
* MySQL version 5.0 or greater

INSTALLATION INSTRUCTIONS

1. First make sure that Elementor Page Builder is installed, As this plugin works only with it.
2. Download the plugin then Upload it to the plugin folder: /wp-content/plugins/ or install it through the WordPress plugins screen directly.
3. Activate the plugin through the ‘Plugins’ screen in WordPress
